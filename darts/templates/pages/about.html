{% trans 'Ons logo en inspiratie' %}

{% blocktrans %} ChudartZ is een unieke samensmelting van de Pokémon- en dartswereld, verrijkt met een speciale Japanse twist. {% endblocktrans %}

  • {% blocktrans %} "Chu" komt van Pikachu, een van de meest iconische Pokémon, en vertegenwoordigt onze sterke band met het Pokémon-universum. {% endblocktrans %}
  • {% blocktrans %} “Z” staat voor de krachtige Z-moves uit de Pokémon-games, die symbool staan voor kracht en dynamiek, vergelijkbaar met onze toewijding aan zowel de darts- als de collectibles wereld. {% endblocktrans %}
  • {% blocktrans %} “DartZ” verwijst natuurlijk naar darten, een sport uit Engeland waar wij een diepe passie voor hebben. {% endblocktrans %}

{% blocktrans %} Ons logo, met de naam 'ChudartZ' in het Japans, vangt de unieke mix van darts en Pokémon, met een Japanse stijl die onze culturele verbondenheid en onderscheidende karakter benadrukt. {% endblocktrans %}

{% trans 'Missie en Visie' %}

{% trans 'Missie' %}

{% blocktrans %} Onze missie is om een plek te creëren in de regio Zedelgem waar jongeren zich kunnen ontwikkelen, hun passie voor darts kunnen ontdekken en zich hierin verder kunnen verfijnen. We streven ernaar om in de dartschool jongeren te inspireren, te begeleiden en op een educatieve manier het individuele niveau omhoog te krikken. Door middel van regelmatige trainingen streven we ernaar zowel technische vaardigheden als persoonlijke groei te bevorderen. {% endblocktrans %}

{% trans 'Visie' %}

{% blocktrans %} Onze visie bij ChudartZ is om een toonaangevende plek voor darts te worden, te beginnen met onze jeugd dartschool en zo door te groeien naar onze dartsclub. We willen een inspirerende omgeving creëren waar respect, doorzettingsvermogen en sportiviteit centraal staan. Ons doel is om een broedplaats voor talent te zijn, jongeren te helpen hun passie te ontdekken en hun ambities te realiseren. Tevens streven we ernaar bij te dragen aan de erkenning van darts als volwaardige sport in België en een sterke gemeenschap rond deze prachtige discipline op te bouwen. {% endblocktrans %}

Dartschool

  1. {% blocktrans %}Ontwikkeling van Basisvaardigheden:{% endblocktrans %}

    {% blocktrans %}Het aanleren van de basisprincipes van darten, zoals{% endblocktrans %}

    • {% blocktrans %}de juiste houding{% endblocktrans %}
    • {% blocktrans %}de ademhaling{% endblocktrans %}

    {% blocktrans %}Dit zorgt ervoor dat leerlingen een solide fundament opbouwen waarop ze hun vaardigheden verder kunnen ontwikkelen.{% endblocktrans %}

  2. {% blocktrans %}Bevorderen van competitief denken:{% endblocktrans %}

    {% blocktrans %} Het stimuleren van een gezonde competitieve geest bij jongeren, waarbij fair play, sportiviteit en zelfverbetering centraal staan. Dit helpt hen om niet alleen beter te worden in darten, maar ook om belangrijke levensvaardigheden te ontwikkelen. {% endblocktrans %}

  3. {% blocktrans %}Persoonlijke Groei en zelfvertrouwen:{% endblocktrans %}

    {% blocktrans %} Het trainen van talentvolle spelers om deel te nemen aan lokale, nationale en wie weet op termijn internationale dartstoernooien. {% endblocktrans %}

  4. {% blocktrans %}Voorbereiden op Competitieve Darts:{% endblocktrans %}

    {% blocktrans %} Het stimuleren van een gezonde competitieve geest bij jongeren, waarbij fair play, sportiviteit en zelfverbetering centraal staan. Dit helpt hen om niet alleen beter te worden in darten, maar ook om belangrijke levensvaardigheden te ontwikkelen. {% endblocktrans %}

  5. {% blocktrans %}Creëren van Gemeenschapsgevoel:{% endblocktrans %}

    {% blocktrans %} Het opbouwen van een hechte en ondersteunende gemeenschap waar spelers van alle leeftijden en niveaus elkaar kunnen inspireren, ondersteunen en samen plezier kunnen hebben. Dit bevordert een gevoel van samenhorigheid en gedeelde passie. {% endblocktrans %}

  6. {% blocktrans %}Educatieve en Leuke Activiteiten:{% endblocktrans %}

    {% blocktrans %} Het aanbieden van educatieve workshops, toernooien en andere activiteiten om het leren leuk te maken en tegelijkertijd de vaardigheden en kennis van de deelnemers te vergroten. {% endblocktrans %}

  7. {% blocktrans %}Mentorschap:{% endblocktrans %}

    {% blocktrans %} Het bieden van mentorschap aan jongere spelers door ervaren dartspelers (binnen de school), wat niet alleen hun dartvaardigheden verbetert, maar hen ook leert om te groeien binnen de dartsgemeenschap. {% endblocktrans %}

  8. {% blocktrans %}Fysieke en mentale gezondheid bevorderen:{% endblocktrans %}
    • {% blocktrans %} Het bevorderen van zowel de fysieke als mentale gezondheid door het aanmoedigen van regelmatige fysieke activiteit, concentratie, en stressmanagement via darten. {% endblocktrans %}
    • {% blocktrans %} Hoofdrekenen is een essentieel onderdeel van het darten, waardoor je niet alleen je rekenvaardigheden aanscherpt, maar ook je algemene concentratie en strategisch inzicht verbetert. {% endblocktrans %}

{% blocktrans %} Met deze doelen biedt een dartschool niet alleen technische training, maar ook een platform voor persoonlijke en sociale ontwikkeling. {% endblocktrans %}

{% blocktrans %} Voeg daar nog een dosis theorie met specifieke dartsterminologie aan toe, en voor je het weet, schitter je op toernooien. {% endblocktrans %}

{% blocktrans %} Met alle aangereikte tools en het ter beschikking stellen van de accommodatie zijn jullie nu zelf aan zet. Want het pijltje zal uiteindelijk door jullie zelf gegooid moeten worden.

Succes! {% endblocktrans %}
