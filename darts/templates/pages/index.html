{% trans 'Over ChudartZ' %}

{% blocktrans %} Welkom bij ChudartZ, dé bestemming voor darts in Zedelgem en daarbuiten. We willen onze passie voor darts graag met jullie delen en overbrengen. Onze eigen jeugd dartschool biedt spelers van alle niveaus de kans om hun vaardigheden te verbeteren, met een bijzondere focus op het ontwikkelen van jong talent. We streven ernaar om regelmatig toernooien te organiseren waarin spelers van alle niveaus samenkomen en de competitie met elkaar aangaan. {% endblocktrans %}

{% trans 'Lees Meer' %}

{% trans 'Dartschool' %}

{% blocktrans %} Of je nu een beginner bent of al ervaren aan de oche staat, bij Chudartz kun je jouw dartvaardigheden naar een hoger niveau tillen. Onze dartschool biedt begeleiding op maat voor jeugdspelers van alle niveaus. Ontdek jouw talent, verbeter je techniek en wie weet schitter jij straks op het podium! {% endblocktrans %}

{% trans 'Toernooien' %}

{% blocktrans %} Doe mee aan onze darttoernooien! Of je nu een beginner bent of een doorgewinterde professional, onze darttoernooien staan voor jou open. Een geweldige kans om je skills te testen, plezier te maken én mooie prijzen te winnen. Schrijf je vandaag nog in en mis de actie niet! {% endblocktrans %}

{% trans 'Teambuilding' %}

{% blocktrans %} Op zoek naar een originele en sportieve teambuilding? Bij ChudartZ organiseren we dynamische dartsevents voor bedrijven, waar plezier, competitie en samenwerking centraal staan. Perfect voor alle ervaringsniveaus! Versterk de teamspirit en beleef een onvergetelijke dag met je collega's. Boek nu je teambuilding! {% endblocktrans %}

{% trans 'Verkoop van Darts' %}

{% blocktrans %} Bij Chudartz kunt u ook terecht voor een breed scala aan darts producten, van dartpijlen en borden tot accessoires. Ons assortiment is zorgvuldig samengesteld om aan de behoeften van elke darts-enthousiasteling te voldoen. {% endblocktrans %}

{% trans 'Teambuildings & Workshops' %}

{% blocktrans %} Bij ChudartZ weten we dat geweldige prestaties op de dartbaan niet alleen afhangen van individuele vaardigheden, maar vooral van de kracht van een hecht team. Daarom organiseren wij ook teambuilding-evenementen. {% endblocktrans %}

1. {% blocktrans %}Hoe kan ik me inschrijven voor het toernooi?{% endblocktrans %}

{% blocktrans %}Inschrijven kan via {% endblocktrans %}{% blocktrans %}onze website{% endblocktrans %}{% blocktrans %} of ter plaatse op de dag van het evenement, afhankelijk van beschikbaarheid{% endblocktrans %}

2. {% blocktrans %}Wat zijn de dartregels ?{% endblocktrans %}

{% blocktrans %}Onze dartregels kan u {% endblocktrans %}{% blocktrans %}hier{% endblocktrans %} {% blocktrans %}terugvinden.{% endblocktrans %}

3. {% blocktrans %}Is de dartschool toegankelijk voor mensen met een beperking?{% endblocktrans %}

{% blocktrans %}Hoewel we er alles aan doen om een gastvrije omgeving te bieden, willen we erop wijzen dat onze huidige locatie helaas niet volledig toegankelijk is voor mindervaliden vanwege de structuur van het gebouw. We waarderen uw begrip en werken eraan om in de toekomst meer toegankelijke faciliteiten te bieden.{% endblocktrans %}

4. {% blocktrans %}Is er een leeftijdsbeperking om lid te worden van de school of deel te nemen aan een toernooi?{% endblocktrans %}

Onze school en toernooien staan open voor iedereen vanaf 10 jaar. In uitzonderlijke gevallen kunnen jongere talenten zich ook inschrijven. Ouderlijke toestemming is hierbij altijd vereist. Bij verdere vragen, neem gerust contact met ons op.

5. {% blocktrans %}Wat is de prijzenpot voor de winnaars?{% endblocktrans %}

{% blocktrans %}De prijzenpot verschilt per toernooi, maar bevat vaak geldprijzen, trofeeën en andere leuke verrassingen.{% endblocktrans %}

6. {% blocktrans %}Kan ik mij op de dag zelf nog inschrijven voor een toernooi?{% endblocktrans %}

{% blocktrans %}Afhankelijk van de beschikbare plaatsen kun je je op de dag zelf nog inschrijven. We raden echter aan om vooraf te registreren om zeker te zijn van deelname.{% endblocktrans %}
